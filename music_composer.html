Music Composer

This was a project made with Java on my first year. I have just learned Java and it was particularly hard solely because of the GUI components, since I only coded the GUI components without any external programs. It was a really good learning experience, and signal manipulation for different instruments and notes was a fun challenge.
On this simple program, you could compose your music, listen to it, change the tempo the brush properties of the notes and their loudness. The program allows the user to work on different canvases for different instruments. It also has a feature where you can select different scales, which will black out certain note rows for in-scale composition. This feature is important since the program is aimed at beginners and is very simple to use.
